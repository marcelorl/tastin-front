### Overview

App to find restaurants around the user.

### Tools
  - Atomic design
  - Create React App
  - React
  - Redux
  - Semistandard
  - Auth0
  - google-map-react
  - material-ui
  - enzyme
  - react-devtools
  - tests
  - test coverage report
  
### Commands
  #### Installation
    - cp .env.example .env (fill with the right variables)
    - yarn
  #### Development
    - yarn start
    - NOTE: to access it locally http://localhost:3000
  #### Testing
    - yarn test
    - yarn coverage
  #### Deploy
    - yarn deploy
